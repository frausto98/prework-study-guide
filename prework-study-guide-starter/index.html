HTML

  • The head element contains information about the page.
  • The body element represents the visible content shown to the user.
  • The section element is within the body element.
  • When one element contains many other elements inside, it's considered a parent/child relationship.

CSS

  • Margin indicates how much space we want around the outside of an element.
  • Padding indicates how much space we want around the content inside the element.
  • Declarations contain the CSS property we want to apply, and the value of said property.
  • You can combine two selectors in a single delcaration by seperating them with a comma.

Git

  • git status: checks what branch you're currently in.
  • git checkout -b "branch name": creates a new branch and switches to it.
  • git checkout main: is a reliable way to get to the main branch.
  • git has many commands that will be used, but knowing how to use push and pull is essential to saving your work.

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer excutes code in a script.
  • A function is instructions that tells the computer how to perform a certain task.
  • When using an if statement, the code will only execute IF the statement meets a condition or is true.
